# Training Planner

**Автор:** Алмаз Газизов

## Описание
Приложение для записи и планирования спортивных тренировок. Позволяет сохранять дату, тип активности и время выполнения в JSON-файл.

## Функции
- Добавление тренировок с валидацией данных.
- Фильтрация списка по типу тренировки в реальном времени.
- Автоматическое сохранение данных между сессиями.

## Примеры использования (Тесты)
1. **Успешный ввод:** Дата: `01.05.2024`, Тип: `Бег`, Длительность: `30`. Результат: запись добавлена.
2. **Ошибка валидации:** Дата: `2024-01-01` или Длительность: `-5`. Результат: всплывающее окно с ошибкой.
